# PacMan-Assembly
Pac-Man simplificado em Assembly – Arquitetura de Computadores

Projeto desenvolvido no âmbito da disciplina de Arquitetura de Computadores.

## Descrição
Este projeto implementa a base gráfica de um jogo Pac-Man simplificado, utilizando linguagem assembly
no simulador PEPE. O foco principal é a manipulação direta da PixelScreen (32x32 pixels), o uso de
endereçamento mapeado em memória e a organização modular do código.

Nesta fase, o programa realiza:
- Inicialização das posições dos objetos;
- Desenho das comidas nos quatro cantos do ecrã;
- Desenho dos inimigos;
- Rotinas genéricas de desenho de pixels.

## Estado do Projeto
O projeto encontra-se numa fase inicial, servindo como base para futuras extensões, nomeadamente:
- Introdução de ciclo principal do jogo;
- Leitura de teclado;
- Movimento do Pac-Man e dos inimigos;
- Uso de interrupções para temporização.

## Estrutura
- `pacman.asm` — código-fonte em assembly do projeto.
